## Setup SDL2 (Ubuntu)
```
$ sudo apt-get install libsdl2-dev
$ sudo apt-get install libsdl2-image-dev libsdl2-mixer-dev libsdl2-net-dev
```

## Build & Run
```
$ sudo bash build.sh
$ ./t_rex_runner.out
```
